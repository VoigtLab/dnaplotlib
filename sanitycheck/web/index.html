Home

On this page, the link to the GitHub repository is provided. DNAplotlib is primarily intended for python-based user interface, and we hope that users also contribute new code to the repository to make this tool more broadly useful.


Single design

For quick, single designs, click to choose part colors and part types to build a string that will be converted to a DNAplotlib genetic diagram.


Library design

For larger designs and library-based designs, parts and the composition of those parts into a design can be specified by uploading CSV files. Optional files describe part customization and regulatory information (activation/repression).


Examples

Here we provide three examples to introduce you to the functionality of DNAplot lib. In the first example, all available parts are specified. In the second example, certain parts have customizations such as height, width, linewidth, spacing, and arrowhead styles. In the third example, library-based designs can be built. These designs motivated the development of DNAplotlib.
